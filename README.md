# InflationForecast
7 farklı regresyon modeli ve 18 adet ekonomik değişken kullanarak Türkiye'nin gelecek 12 aylık enflasyonunu tahmin etmeye çalışıyoruz.
Kullanılan Modeller:
1) Lineer Regresyon

2) Bayesian Regresyon

3) Gaussian Regresyon

4) Kernel Regresyon

5) Lasso Regresyon

6) Lars Regresyon

7) SGD Regresyon

8) SARIMAX

9)Prophet
